## Despliegue

### Frontend

El frontend fue desarrollado con React y desplegado en Vercel.  
El repositorio fue conectado a Vercel, que detectó automáticamente el proyecto y realizó el build y despliegue.  
Se configuró la variable de entorno con la URL del backend.

### Backend

El backend fue desplegado en Railway a partir de un repositorio de GitHub.  
Se configuraron las variables de entorno necesarias y Railway realizó el despliegue automático, generando una URL pública.
